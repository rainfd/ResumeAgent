# Resume Assistant TUI 使用指南

## 🚀 启动应用

```bash
# 使用虚拟环境运行
./venv/bin/python demo_tui.py

# 或者激活虚拟环境后运行
source venv/bin/activate
python demo_tui.py
```

## 📱 交互模式

Resume Assistant TUI 支持两种交互模式：

### 🎯 浏览模式（默认）
- **特点**: 快速单字符响应，类似vim的操作方式
- **提示**: `浏览模式 (1-6切换面板, :命令模式, q退出)`
- **适用场景**: 日常快速浏览和操作

**支持的快捷键:**
- `1-6`: 直接切换到对应面板（无需回车）
- `j` / `k` / `↑` / `↓`: 滚动内容
- `h`: 显示帮助信息
- `r`: 刷新当前界面
- `q`: 退出应用
- `:`: 进入命令模式

**简历管理专用操作（在面板3中）:**
- `u`: 上传新简历
- `v`: 查看简历详情
- `d`: 删除简历

**AI分析专用操作（在面板4中）:**
- `s`: 开始新的AI分析
- `v`: 查看分析详情
- `d`: 删除分析记录

### ⌨️ 命令模式
- **特点**: 传统命令行交互，需要回车确认
- **提示**: `命令模式 (输入命令并按回车确认, 或输入'esc'退出)`
- **适用场景**: 执行复杂命令或在受限环境中使用

**进入方式**: 在浏览模式下按 `:`

**支持的命令:**
- `1-6`: 切换到对应面板
- `q`: 退出应用
- `r`: 刷新界面
- `esc` 或 `escape`: 退出命令模式

## 📋 面板功能

### 1. 🏠 主页 (按1)
- 显示应用简介和使用说明
- 功能特性概览

### 2. 💼 职位管理 (按2) 
- 查看职位列表
- 管理目标职位信息
- *开发中: 添加BOSS直聘职位链接*

### 3. 📄 简历管理 (按3) - **Task2核心功能**
- **`u`**: 上传简历文件（支持PDF和Markdown）
- **`v`**: 查看简历详情和内容预览
- **`d`**: 删除选中的简历
- 显示已上传简历的统计信息

### 4. 🤖 AI分析 (按4) - **Task3核心功能**
- **`s`**: 开始新的AI分析（选择简历和职位进行匹配）
- **`v`**: 查看历史分析详情和结果
- **`d`**: 删除分析记录
- 显示分析历史和匹配度评分
- 支持DeepSeek API集成和模拟模式

### 5. 💬 打招呼语 (按5)
- 查看生成的个性化打招呼语
- 不同版本的开场白预览
- *开发中: 基于简历和职位生成*

### 6. ⚙️ 设置 (按6)
- 查看和修改应用配置
- 系统设置管理

## 📄 简历上传功能详解

### 支持的格式
- **PDF文件** (.pdf): 自动提取文本内容
- **Markdown文件** (.md, .markdown): 直接读取内容
- **纯文本文件** (.txt): 基础文本支持

### 上传流程
1. 在简历管理面板（按3）
2. 按 `u` 键启动上传
3. 输入文件路径（支持相对路径和绝对路径）
4. 系统自动解析并存储简历
5. 显示处理结果和文件统计信息

### 管理操作
- **查看详情**: 按 `v` 键选择简历查看完整信息
- **删除简历**: 按 `d` 键选择并确认删除
- **内容预览**: 查看简历时显示前500字符预览

## 🤖 AI分析功能详解

### 分析功能
- **智能匹配**: 分析简历与职位的匹配度
- **多维评分**: 从技能、经验、教育、契合度等维度评估
- **优化建议**: 提供具体的改进建议
- **技能对比**: 识别匹配技能和缺失技能

### 使用流程
1. 在AI分析面板（按4）
2. 按 `s` 键开始新分析
3. 选择要分析的简历文件
4. 选择目标职位
5. 查看详细的分析结果和建议

### 分析结果
- **总体匹配度**: 0-100分的综合评分
- **分项评分**: 技能、经验、教育等细分维度
- **优势分析**: 突出的优点和竞争力
- **改进建议**: 具体的优化方向
- **技能清单**: 匹配和缺失的技能对比

### 运行模式
- **AI模式**: 配置DeepSeek API密钥后使用真实AI分析
- **模拟模式**: 未配置API时使用示例分析结果
- **历史管理**: 分析结果持久化存储，支持查看和删除

## 🔧 环境兼容性

### 完整终端环境
- 支持单字符直接响应
- vim风格的j/k滚动
- 最佳用户体验

### 受限环境（如某些IDE终端）
- 自动检测并切换到简化模式
- 使用传统input()交互
- 功能完全可用，仅交互方式略有不同

## 🐛 故障排除

### ModuleNotFoundError: No module named 'rich'
```bash
# 确保使用虚拟环境
./venv/bin/python demo_tui.py
# 或安装依赖
pip install rich
```

### "Inappropriate ioctl for device"
- 这是正常现象，应用会自动切换到简化模式
- 功能不受影响，仅交互方式改变

### 简历上传失败
- 检查文件路径是否正确
- 确认文件格式受支持（pdf, md, txt）
- 查看错误提示信息进行相应处理

## 💡 使用技巧

1. **快速导航**: 记住数字键1-6对应的面板，可以快速切换
2. **简历管理**: 建议使用相对路径上传文件，如 `./resume.pdf`
3. **内容滚动**: 在长内容面板中使用j/k键快速浏览
4. **命令模式**: 在需要输入复杂命令时使用，支持完整的命令输入
5. **帮助信息**: 按h键随时查看快捷键帮助
